# Trabajando con Create Vite (ex Vue Cli)

<https://cli.vuejs.org/#getting-started>
<https://github.com/vuejs/create-vue>


```sh
npm create vue@latest ./
```

1. No eligir nada [none] (enter)
2. No eligir nada [none] (enter)
3. Dejar todo como está [no] (enter) -> Genera el código demo.
4. npm install (Instalamos las dependencias)
5. npm run dev (arracamos el servidor de desarrollo)
